
## STEP 0 — Initialize Repository (Bootstrap)

### Objective

Create the Next.js project skeleton with TypeScript, install core dependencies, and prepare the folder structure.

### Prerequisites

- Node.js >= 18.0.0 installed
- npm or yarn package manager
- Terminal access

### Actions

#### 1. Create Next.js project

```bash
npx create-next-app@latest chatcommerce-ai \
  --typescript \
  --eslint \
  --app \
  --src-dir=false \
  --tailwind=false \
  --use-npm
```

**Configuration prompts:**

- TypeScript: Yes
- ESLint: Yes
- Tailwind CSS: No
- `src/` directory: No
- App Router: Yes
- Import alias: Yes (@/\*)

#### 2. Navigate to project directory

```bash
cd chatcommerce-ai
```

#### 3. Install AI SDK and dependencies

```bash
npm install ai @ai-sdk/openai @supabase/supabase-js
```

**Package versions (reference)**:

- `ai`: Latest (3.x+)
- `@ai-sdk/openai`: Latest
- `@supabase/supabase-js`: Latest (2.x+)

#### 4. Create folder structure

```bash
mkdir -p app/api/chat lib types
```

**Expected structure:**

```
chatcommerce-ai/
├── app/
│   └── api/
│       └── chat/
├── lib/
└── types/
```

### Validation

- [ ] Project directory `chatcommerce-ai` exists
- [ ] `package.json` contains all dependencies
- [ ] Folders `app/api/chat`, `lib`, `types` exist
- [ ] Development server starts successfully

**Validation commands:**

```bash
# Check Node version
node --version  # Should be >= 18

# Verify installation
ls -la chatcommerce-ai/

# Test dev server
npm run dev  # Should start on http://localhost:3000
```

### Troubleshooting

**Problem**: `npx create-next-app` fails with permission error
**Solution**: Run with sudo or fix npm permissions:

```bash
sudo chown -R $(whoami) ~/.npm
```

**Problem**: Port 3000 already in use
**Solution**: Kill existing process or use different port:

```bash
lsof -ti:3000 | xargs kill
# OR
PORT=3001 npm run dev
```

**Problem**: TypeScript errors on first run
**Solution**: This is normal; ignore for now, they'll be resolved in subsequent steps.

### Expected Outputs

Terminal should show:

```
✔ Creating a new Next.js app in /path/to/chatcommerce-ai
✔ Installing dependencies...
✔ Initializing project...

Success! Created chatcommerce-ai at /path/to/chatcommerce-ai
```

Browser at `http://localhost:3000` should display default Next.js landing page.
